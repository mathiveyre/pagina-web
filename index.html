Cómo hice este sitio web

Este sitio está construido con código HTML y CSS puro, sin frameworks ni librerías externas. Todo el contenido está en un solo archivo llamado index.html.

1. Estructura básica

Usé etiquetas estándar como <html>, <head>, y <body> para definir la estructura del documento.

2. Estilo minimalista

El estilo se definió dentro de una etiqueta <style> en el encabezado. Se usó una fuente sans-serif, colores neutros, y márgenes amplios para lograr un diseño limpio.

3. Responsive

Agregué meta viewport para que el diseño se vea bien en móviles:

<meta name="viewport" content="width=device-width, initial-scale=1">

4. Hospedaje

Este archivo puede subirse directamente a GitHub Pages para que esté disponible en Internet.

5. Resultado

Un sitio rápido, simple, sin distracciones, ideal para explicar conceptos o mostrar portafolios básicos.

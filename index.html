¿Cómo hice este sitio?

Este sitio está hecho con HTML y CSS puro, siguiendo un enfoque minimalista. A continuación te muestro cómo podés crear algo similar desde cero.

1. Estructura HTML básica

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi sitio</title>
  </head>
  <body>
    <h1>Hola mundo</h1>
  </body>
</html>

2. Estilo con CSS

<style>
  body {
    font-family: sans-serif;
    background: #fafafa;
    color: #333;
    padding: 40px;
    max-width: 600px;
    margin: auto;
  }
</style>

3. Guardá y abrí

Guardá el archivo como index.html y abrilo en tu navegador para ver el resultado.

4. Publicalo

Subilo a GitHub Pages y tenés tu web online gratis.
